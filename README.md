# homebrew-chuch

Personal Homebrew tap for `chuch-term`.

## Install

```bash
brew tap KrzysPawlo/chuch
brew install chuch-term
```

## Upgrade

```bash
brew update
brew upgrade chuch-term
```

## Repository Layout

- `Formula/chuch-term.rb` — Homebrew formula for `chuch-term`

Main project repository:
